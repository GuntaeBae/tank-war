# Tank War (탱크 전쟁) 🚀

HTML5 Canvas와 JavaScript로 구현된 고전 아케이드 스타일의 탱크 슈팅 게임입니다.

## 🤖 Vibe Coding (바이브 코딩)

이 프로젝트는 **Google Gemini**와 **VS Code**를 활용한 **바이브 코딩(Vibe Coding)** 방식으로 개발되었습니다.

*   **AI-Assisted Development**: Gemini Code Assist를 통해 게임 로직 설계, 버그 수정, 기능 확장을 진행했습니다.
*   **Efficiency**: 자연어 프롬프트를 통해 코드를 생성하고, 실시간으로 피드백을 주고받으며 개발 속도를 높였습니다.

## 🎮 게임 방법

적 탱크의 공격을 피하고, 포탄을 발사하여 적을 제압하세요!

### 조작법
*   **이동**: 방향키 (`↑`, `↓`, `←`, `→`) 또는 `W`, `A`, `S`, `D`
*   **발사**: 스페이스바 (`Space`)
*   **재시작**: 게임 오버 시 `Enter` 또는 `R` 키 (구현 여부에 따라 다름)

## 🛠 기술 스택

*   **Core**: HTML5, CSS3
*   **Logic**: Vanilla JavaScript (ES6+)
*   **Rendering**: HTML5 Canvas API

## 📦 실행 방법

별도의 설치 과정 없이 웹 브라우저만 있으면 실행 가능합니다.

1. 이 저장소를 다운로드하거나 클론(Clone)합니다.
2. 프로젝트 폴더 내의 `index.html` 파일을 더블 클릭하여 웹 브라우저에서 엽니다.
3. 게임을 즐기세요!

## ✨ 주요 기능

*   플레이어 및 적 탱크의 부드러운 이동
*   충돌 감지 시스템 (탱크 간, 포탄과 탱크)
*   점수 기록 및 게임 상태 관리
